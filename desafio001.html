Entendendo o Desafio

A ideia deste Desafio de Projeto é elaborar um website, utilizando uma estrutura bem básica, utilizando tags aprendidas durante as aulas práticas:



Utilizar todas as tags explicadas nas aulas:

h1 >>> Título 1 de maior valor hierárquico

h2 >>> Título 2

h3 >> Título 3

h4 >>> Título 4

h5 >>> Título 5

h6 >>> Título 6 de menor valor hierárquico

p >>> Principal tag de parágrafo

mark >>> Representa um trecho de destaque em um texto

small >>> Define textos menores

i >>> Transforma o conteúdo em itálico

u >>> Tag de sublinhado

strong >>> Importância

ol >>> Lista ordenada

ul >>> Lista não ordenada

li >>> Lista

a >>> Tag âncora para um link

hr >>> Colocar uma linha na página

sub >>> Define um texto que é exibido como um texto subscrito

sup >>> Define um texto que representa como texto sobrescrito

blockquote >>> Conteúdo de outra fonte



Utilizar novas tags sugeridas:

font >>> Alterar o tamanho da fonte, face, cor, etc.

del >>> Texto excluído do documento

p >>> Principal tag de parágrafo

abbr >>> Abreviação

(a ideia é buscar estas tags na internet, entender como ela funciona e utilizar no texto).
